What is Node-Js?

  • Node-Js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside of a web browser. It allows developers to use JavaScript on the server-side to build fast, scalable, and high-performance applications.

  • Open-source: Node.js is freely available and can be used by anyone for any purpose.

  • Cross-platform: It runs on Windows, macOS, Linux, and other platforms.

  • JavaScript runtime: It is a JavaScript runtime environment that allows developers to execute JavaScript on the server-side.

  • Non-blocking I/O: Node.js uses an event-driven, non-blocking I/O model which makes it fast and efficient.

Use of Node-Js in Web :

  1. Built on Chrome's V8 engine: The V8 JavaScript engine is responsible for executing JavaScript code in Node.js.

  2. Web application development: Node.js is commonly used for building web applications, both server-side and client-side.

  3. etwork applications: It is also used for building network applications, such as web servers, proxies, and chat applications.
