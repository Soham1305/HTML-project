What is React-Js ?

  • React -Js is a JavaScript library for building user interfaces.

  • It allows developers to build web applications that can update

  • and render efficiently in response to changes in data, without requiring a full page reload. React uses a virtual DOM

  • optimizes updates by only changing the parts of the UI that have actually changed
