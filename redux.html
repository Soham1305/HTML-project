What is Redux ?

  • Redux is a state management library for JavaScript applications.

  • Single source of truth: Redux stores the entire state of the application in a single, immutable state tree, which acts as a single source of truth for the application.

  • State immutability: The state tree can only be modified through the dispatch of actions, which are plain JavaScript objects describing the desired change to the state.

  • Pure functions: Reducers, which are responsible for modifying the state tree, are pure functions that take the previous state and an action, and return the next state.

  • Redux can be viewed and edited using a text editor, such as Notepad or Sublime Text, and can be previewed in a web browser.

Use of Redux in Web :

  1. Predictable state updates: By using pure functions to update the state, Redux makes it easy to understand how the state of the application changes over time, which is useful for debugging.

  2. Centralized store: Redux uses a centralized store to hold the state, making it easy to share data between different parts of the application.

  3. Middleware: Redux provides a middleware mechanism to extend the behavior of the store, allowing developers to add custom functionality, such as logging or asynchronous actions.
