What is HTML 5 ?

  • HTML 5 of elements, which are represented by tags, enclosed in angle brackets .

  • HTML documents are structured into head and body sections, with the head section including meta information such as title and description, and the body section containing the main content of the page.

  • HTML supports various types of media such as text, images, videos, and audio, which can be incorporated into a web page using tags such as img,audio,videoHTML also provides a means for creating links to other pages and resources using the a tag.

  • HTML includes support for styling and presentation using Cascading Style Sheets (CSS) and JavaScript, which can be applied to elements on a web page to change their appearance and behavior.

  • HTML can be viewed and edited using a text editor, such as Notepad or Sublime Text, and can be previewed in a web browser.

Use of HTML 5 in Web :

  1. Enhanced Semantic Markup: HTML5 introduces new elements, such as header, nav, footer, and article, that provide greater semantic meaning to the content of a web page, making it easier for search engines and assistive technologies to understand the structure and content of a page.

  2. Improved Multimedia Support: HTML5 provides new elements, such as video and audio, for embedding multimedia content directly into a web page, without the need for external plugins such as Flash

  3. Offline Capabilities: HTML5 introduces the concept of offline web applications, allowing web pages to work without an internet connection and providing a more native-like experience for users. This is achieved through the use of technologies such as the Cache API and Service Workers.
