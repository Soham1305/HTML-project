What is CSS ?

  • HTML 5 of elements, which are represented by tags, enclosed in angle brackets .

  • CSS provides a way to separate the presentation of a web page from its content, allowing for greater control and flexibility in designing the look and feel of a site.

  • CSS styles can be applied to specific HTML elements using selectors, such as classes, IDs, and element types, with the styles defined in a stylesheet file, linked to the HTML page with a Link tag.

  • CSS provides various styling properties, such as color, font, size, padding, margin, and border, that can be used to control the appearance of elements.

  • CSS can be viewed and edited using a text editor, such as Notepad or Sublime Text, and can be previewed in a web browser.

Use of CSS in Web :

  1. isual Design: CSS is used to control the visual appearance of a web page, such as the font, color, size, spacing, and layout of elements. This allows designers to create consistent and appealing designs that match a brand's style and identity.

  2. esponsive Design: CSS is used to make web pages adjust to different screen sizes and devices, through the use of media queries and flexible units like percentages and viewport units. This provides a better user experience for users on different devices, with the same content being presented in a way that is optimized for each screen size.

  3. Interactivity and Animation: CSS is used to add dynamic and interactive elements to a web page, such as hover effects, transitions, and animations. This can make a web page more engaging and user-friendly, and provide visual cues and feedback to users as they interact with a page.
